# Project CemQ_Predicts 

- Bajo la premisa de **'Construir progreso para las personas y el planeta'** Holcim en el marco de sus estrategias de sostenibilidad se propone crecer de la mano de retos en descarbonización y el uso de tecnologias aplicables a la industria, es por ello que este proyecto prente un acercamiento a los datos disponibles en los sistemas de información actuales con el fin de poder validar y examinar posibles alternativas en la mejora del proceso de producción.

# Contenido
Contiene las siguientes carpetas y archivos:

* `data`: Datos crudos obtenidos del sistema de información.
* `src`: código o implementación del proyecto en Python.
* `docs`: plantillas de los documentos definidos en la metodología.
* `scripts`: scripts/notebooks que se ejecutarán.
* `pyproject.toml`: archivo de definición del proyecto en Python.
